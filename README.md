# Valentina’s Flowers Paradise

## 1. Project Overview

Valentina’s Flowers Paradise is a simple, responsive website inspired by my wife Valentina’s passion for flowers. The site includes a homepage, a gallery, an about page, and a contact page. It is designed to be easy to navigate and accessible on all devices.

## 2. Purpose of the Project

The aim of this project is to build a clean and user‑friendly static website using HTML, CSS and Bootstrap. The site presents floral images, basic information, and a clear layout.

## 3. User Stories

#### Must Have 

* As a user, I want to navigate the website easily so I can find the information I need.
* As a user, I want the website to work on mobile, tablet, and desktop so I can view it on any device.
* As a user, I want clear sections (Home, Gallery, About, Contact) so I can understand the structure of the site.

#### Should Have

* As a user, I want the gallery images to be displayed in a clean and organised layout so I can enjoy browsing them.

#### Could Have

* As a user, I want the navigation bar to remain visible at the top of the page while I scroll, so I can access the menu at any time.

#### Wont't Have

* No JavaScript features (except the Bootstrap navbar behaviour).
* No backend, database, or interactive systems.

## 3. Features

* Responsive navigation bar
* Hero section with title and subtitle
* Image gallery
* About section
* Contact page
* Sticky navigation bar (CSS only)
* Favicon in multiple sizes

### Future Features
